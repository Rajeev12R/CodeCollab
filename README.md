# CodeCollab 🚀

**CodeCollab** is a collaborative web application that helps developers work together on projects using GitHub. It allows team members to contribute through individual branches and merge their work efficiently into the main project.

## 🔥 Features

- GitHub OAuth login
- Invite and add collaborators
- Track contributions and activity
- Clean and organized code merging

## 🎯 Project Goal

Our goal is to build a platform where developers can experience a real-world GitHub workflow — including branching, pull requests, code reviews, and collaboration.

## ⚙️ Tech Stack

- **Frontend**: React.js, Tailwind CSS  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Others**: GitHub API, OAuth

## 🌟 Unique Feature: Guided Git Workflow

CodeCollab is not just another collaboration tool — it provides a **step-by-step Git workflow guide** designed especially for beginners and teams working together.

### 👣 How It Works:

- **Step 1** → Create a new branch using a simple UI  
- **Step 2** → Make changes and understand what to commit  
- **Step 3** → Learn how to stage and commit files  
- **Step 4** → Push your code with visual instructions  
- **Step 5** → Open a Pull Request with confidence

> This interactive workflow makes Git collaboration feel like a guided tutorial — no more confusion, just clean and confident teamwork!

This makes CodeCollab not just a project manager, but a learning companion for every team member. 💻✨
